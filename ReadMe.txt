# Spotify Playlist Recommender

This project is a Flask web application that recommends songs based on a user's Spotify playlist using the OpenAI API.

## Description

The application allows users to log in with their Spotify account, select one of their playlists, and receive song recommendations generated by OpenAI's GPT-4 model based on the tracks in the selected playlist.

## Installation

Follow these steps to set up the project:

### Prerequisites

- Python 3.7 or higher
- pip (Python package installer)
- Spotify Developer Account (for API keys)
- OpenAI API key
- Ensure to set up the .env file

### Usage
Home Page:
- On the home page, click "Authorize" to log in with your Spotify account.
Spotify Authorization:
- You will be redirected to Spotify's login page. Log in with your Spotify account and authorize the application.
Select Playlist:
- After authorization, you will be redirected back to the application. Select one of your playlists from the list displayed.
Get Recommendations:
- Click "Get Recommendations" to receive song recommendations based on the selected playlist. The recommendations will be displayed along with explanations.
Logout:
- Click "Logout" to end your session.
